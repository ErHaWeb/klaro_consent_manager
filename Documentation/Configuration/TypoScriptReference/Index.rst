TypoScript reference
====================

The :ref:`Site Set <configuration-site-set>` and the Static TypoScript Include
both load the TypoScript files from `Configuration/Sets/KlaroConsentManager/`.

Use :ref:`Site Settings <configuration-site-set>` in TYPO3 v13/v14 site
configurations whenever possible. Use :ref:`classic TypoScript constants
<configuration-typoscript-constants>` only when the site still relies on
TypoScript records.
