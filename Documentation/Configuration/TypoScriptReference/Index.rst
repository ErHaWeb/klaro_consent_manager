TypoScript reference
====================

The Site Set and the Static TypoScript Include both load the TypoScript files
from `Configuration/Sets/KlaroConsentManager/`.

Use Site Settings in TYPO3 v13/v14 site configurations whenever possible. Use
classic TypoScript constants only when the site still relies on TypoScript
records.
