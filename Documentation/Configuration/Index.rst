=============
Configuration
=============

The extension has four configuration surfaces:

*   The :ref:`Site Set <configuration-site-set>` is the recommended integration
    method for TYPO3 v13 and TYPO3 v14.
*   Site Settings or classic TypoScript constants override resource paths and
    selected feature settings.
*   TypoScript setup can filter services and can override selected Klaro
    configuration values.
*   Global extension configuration controls trigger replacement URLs and the
    standalone configuration endpoint.
