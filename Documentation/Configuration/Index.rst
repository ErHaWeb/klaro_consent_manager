=============
Configuration
=============

The extension has four configuration surfaces:

*   The :ref:`Site Set <configuration-site-set>` is the recommended integration
    method for TYPO3 v13 and TYPO3 v14.
*   :ref:`Site Settings and classic TypoScript constants
    <configuration-typoscript-constants>` override resource paths and selected
    feature settings.
*   :ref:`TypoScript setup <configuration-typoscript-setup>` can filter
    services and can override selected Klaro configuration values.
*   :ref:`Global extension configuration <configuration-extension-configuration>`
    controls trigger replacement URLs and the standalone configuration endpoint.
